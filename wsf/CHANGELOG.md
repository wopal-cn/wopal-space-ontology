## [1.34.2] - 2026-04-10

### Changed
- Renamed project namespace from `space-flow` / `gsd` to `wsf` (Wopal Space Flow)
- All commands now use `wsf-` hyphenated prefix instead of `wsf:` colon format
- Renamed `gsd-tools.cjs` to `wsf-tools.cjs`
